# mercenary-reselect prototype prompt

너는 게임 UI/프로토타입 개발을 돕는 프론트엔드 개발자다.
HTML, CSS, JavaScript만 사용해서 단일 파일 프로토타입을 만들어줘.

## 목표

- 고등급 중복 결과에 대한 재선택 UX를 검증한다
- 원본 유지 구조가 허탈감 완화에 도움이 되는지 확인한다
- 랜덤 후보 누적 후 마지막 선택 흐름이 직관적인지 확인한다

## 핵심 규칙

- 원본 용병 유지
- 전환 기회 2회
- 같은 등급 풀 8마리
- 랜덤 후보 누적
- 후보 중복 재등장 없음
- 마지막에 원본 + 후보 중 1개 선택

## 화면 구성

- 현재 획득 용병 카드
- 전환 버튼
- 남은 전환 기회 표시
- 누적 후보 리스트
- 최종 선택 영역
- 선택 결과 표시

## 요구사항

- 단일 `index.html`
- 외부 라이브러리 없이 구현
- 모바일 세로형 카드 UI
- 한글 텍스트 사용
- 실제 서비스 코드보다 UX 검증 우선
