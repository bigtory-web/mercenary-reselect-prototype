# 소환 재선택 시스템 — 문서 안내

이 폴더는 `00~05` 6개 파일만 기준으로 사용한다. 같은 역할의 예전 파일은 모두 이 번호 체계 안으로 통합한다.

## 파일 구성

| 파일 | 역할 | 사용 시점 |
| --- | --- | --- |
| `01_planning.md` | 제출용 메인 기획서 | 기본 기준 문서 |
| `02_dev-spec.md` | 내부 구현 참고 문서 | 개발/구현/검토 |
| `03_analysis.md` | 효율 계산과 해석 메모 | 수치/효율 검토 |
| `04_ui-rules.md` | UI 수정 규칙 | HTML/UI 작업 |
| `05_prompts.md` | 작업 요청 템플릿 모음 | 요청문 작성 |

## 작업별 권장 조합

| 작업 | 같이 볼 파일 |
| --- | --- |
| 제출용 내용 검토 | `01_planning.md` |
| 기획 보완 검토 | `01_planning.md` + `02_dev-spec.md` |
| 효율 계산 검토 | `01_planning.md` + `03_analysis.md` |
| UI 수정 | `prototype/index.html` + `04_ui-rules.md` |
| 요청문 작성 | `05_prompts.md` |

## 운영 기준

- 제출용 문서는 `01_planning.md`만 해당한다.
- `02_dev-spec.md`, `03_analysis.md`는 내부 참고용이며 제출하지 않는다.
- 내부 참고 문서는 상단의 `#메모` 표시로 구분한다.
- 이 폴더 안에는 번호 체계 밖의 중복 문서를 두지 않는다.
- 별도 문체 가이드가 필요하면 외부 파일로 참조하되, 현재 이 폴더 안에는 `style-guide.md`가 없다.
