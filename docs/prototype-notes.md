# mercenary-reselect prototype notes

## 목표

- 고등급 중복 결과에 대한 재선택 UX가 직관적인지 확인
- 원본 유지 구조가 허탈감 완화에 실제로 도움이 되는지 확인
- 후보 누적 리스트가 유저 입장에서 이해 가능한지 확인

## 검증 대상

1. 유저가 `즉시 수령`과 `전환 진행`의 차이를 바로 이해하는가
2. 전환 기회 2회가 지나치게 약하거나 강하지 않은가
3. `원본 + 후보` 중 마지막 1개 선택 흐름이 자연스러운가
4. 후보 누적 구조가 “헛돌렸다”는 느낌을 줄여주는가

## 현재 설정

- 같은 등급 풀: 8
- 전환 기회: 2
- 후보 중복 재등장 없음
- 카드 리스트 UI
- 원본 유지
- 마지막에 원본 포함 후보 중 1개 선택

## 화면 구성

- 현재 획득한 고등급 중복 용병 카드
- 전환 버튼
- 남은 전환 기회 표시
- 누적 후보 카드 리스트
- 최종 선택 영역
- 선택 결과 표시 영역

## 프로토타입 메모

- 이 프로토타입은 실제 밸런스 검증보다 UX 흐름 검증에 가깝다.
- 완전 선택 전환은 제외하고, 제한된 랜덤 후보 축적 구조만 보여준다.
