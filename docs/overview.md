# mercenary-reselect overview

`mercenary-reselect`는 수집형/방치형 RPG에서 발생하는 고등급 중복 획득의 허탈감을 완화하기 위해 설계한 시스템 기획 프로젝트다.

핵심 아이디어는 단순 교환이나 완전 확정 전환이 아니라, `원본 유지 + 랜덤 후보 축적 + 마지막 선택` 구조를 통해 직접 획득 가치와 재선택 기회를 동시에 살리는 데 있다.

## 현재 범위

- 시스템 기획 문서 정리
- UX 검증용 단일 파일 프로토타입
- 전용 프롬프트 정리

## 핵심 키워드

- 고등급 중복
- 재선택 기회
- 직접 획득 가치 유지
- 랜덤 후보 축적
- UX 검증
