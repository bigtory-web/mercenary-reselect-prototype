# 소환 재선택 시스템 docs

이 폴더는 `소환 재선택 시스템` 프로젝트 문서를 정리해둔 공간이다.

## 문서 구성

| 문서 | 역할 | 제출 포함 여부 |
| --- | --- | --- |
| `system-planning-doc-v2.md` | 현재 제출 기준 메인 문서. 시스템 목적, 전체 플레이 흐름, 화면별 구현 흐름, 재화 규칙, 예외 처리, 검증 관점을 한 문서에 정리 | 포함 |
| `system-planning-doc-legacy.md` | 이전 메인 초안. 비교 참고용으로 보관 | 미포함 |
| `development-spec.md` | 내부 구현/테스트 참고 문서. 화면별 UI, 상태 전이, 규칙, 예외 처리, 테스트 세팅을 정리 | 미포함 |
| `efficiency-analysis.md` | 내부 효율 검증 계산 근거. 기대 비용, 비교 가정, 해석 정리 | 미포함 |
| `README.md` | docs 폴더 안내문 | 미포함 |

## 사용 기준

- 현재 제출용 기준 문서는 `system-planning-doc-v2.md`다.
- `system-planning-doc-legacy.md`는 이전 버전으로 남겨둔 비교 참고용 문서다.
- 내부 참고 문서는 상단에 `#메모`를 붙여 제출 본문이 아님을 구분한다.
- PPT나 기타 제출물은 `system-planning-doc-v2.md`를 기준으로 생성한다.
