# mercenary-reselect context

## 출발점

이 프로젝트는 “중복은 기본적으로 성장 자원이지만, 특정 고등급 중복은 감정적으로 허탈하게 느껴질 수 있다”는 문제의식에서 시작했다.

## 검토 과정에서 확인한 것

- 완전 선택 전환은 지나치게 강해질 수 있다.
- 순수 랜덤 전환은 다시 허탈감을 만들 수 있다.
- 전환 결과가 다시 전환 재료가 되면 무한 루프가 생길 수 있다.
- 따라서 `보관함 기반 제한적 전환` 구조가 필요하다.

## 현재 방향

- 고등급 중복은 즉시 확정하지 않는다.
- 임시 보관함에서만 전환을 진행한다.
- 랜덤 후보를 누적한다.
- 최종적으로 원본 포함 후보 중 1개를 선택한다.

## 이후 확장 가능 포인트

- 전환기 수급 구조
- 등급별 전환 조건
- 후보 수와 기회 수의 적정선
- 실제 카드형 UI 감정 체감 검증
